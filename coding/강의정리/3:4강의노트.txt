3/4 강의노트

1. 전역변수 사용법 : 전역변수가 쓰인 코드 전체를 함수로 묶어주면 지역변수가 됨. 그 묶은 함수는 
   '익명함수'가 됨
2. 자바스크립트에서는 함수의 {}안에서 선언된 변수만이 그 함수 안에서 지역변수가 되고, for문, if문 안에서는 지역변수로의 의미를 갖지 않음.
3. 정적유효범위(static scoping) : 사용될 때가 아닌 '정의될 때' 정적유효범위 
4. 함수의 용도 : function a() {} ---> 함수
              a = {
		  b : function() {        ----> b에 담겨진 함수는 '메소드(method)
		  }
		};   
  - 함수는 '값'이기 때문에 다른 함수의 '인자'로 전달 될 수 있다.
  - 또한 변수, 매개변수, 리턴값으로 사용될 수 있다 
     => 함수는 js에서 first-class-citizen(object)
